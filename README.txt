Как использовать?
1) В "Открытие чата" нажимает клавишу соответствующей клавише для открытия чата; 
2) В "Команда" пишем команду, которую мы хотим видеть при нажатии на горячую клавишу;
3) В "Горячая клавиша" нажимаем клавишу, по нажатию которой будет отправляться текст из "Команда";
4) Нажимаем кнопку "Нажмите для запуска".
--Обе кнопки Enter не будут доступны для использования из-за создаваемых ими проблем--

После нажатия на кнопку вы можете использовать биндер. Все, что нужно сделать - это нажать на кнопку.

Для изменения команд нужно поменять её в соответствующей ячейке и 
еще раз нажать кнопку.


Если программа работает некорректно, попробуйте запустить от 
имени администратора.
Программа работает ТОЛЬКО на 64 битной системе.


Если вас что-то не устраивает, то вы всегда можете скачать исходники 
и похимичить с ними.
Также по ссылке ниже вы всегда найдете актуальную версию биндера.
